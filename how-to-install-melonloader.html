GUIDE / STANDARD INSTALL

How to Install MelonLoader

MelonLoader is the universal mod loader for Unity games — it lets you run community-made mods and plugins on top of a game without touching the original files. This is the fast path, using the official automated installer.

Before you start: always download MelonLoader from the official GitHub releases page. Third-party "download" sites for mod loaders are a common way people end up with malware — the real thing is free and open-source.

What you'll need

  • A Unity game installed via Steam, Epic, or standalone
  • Windows 7 / 8 / 10 / 11 (64-bit recommended), with admin rights
  • .NET 6.0 Desktop Runtime — required for Il2Cpp games (most modern Unity titles)
  • The game fully closed before you install or update MelonLoader

Installation

01

Download the installer

Head to the MelonLoader.Installer releases page and download the latest MelonLoader.Installer.exe.

02

Run the installer and select your game

Open the installer. On the AUTOMATED tab, click SELECT and browse to your game's shortcut or its .exe inside the install folder.

03

Choose version and architecture

Two dropdowns: version (leave on latest unless a mod needs an older build) and architecture x86/x64 (leave on auto-detect — it matches your game correctly almost every time).

04

Install

Click INSTALL. The installer copies version.dll, dobby.dll, and a MelonLoader folder into your game's install directory. Takes a few seconds.

05

Launch the game once

Start the game normally. A second window — the MelonLoader console — should appear alongside it (alt-tab if you don't see it right away). If it shows up, MelonLoader is working.

[MelonLoader] v0.7.0
Initializing...
Loaded 0 Mods, 0 Plugins
Ready.

This first launch also creates three folders inside your game directory:

FolderWhat goes here
ModsMod .dll files
MelonLoader/PluginsPlugin .dll files
UserDataConfig files, save data for mods
06

Add mods

Drop mod files into Mods (or Plugins for plugins) and relaunch the game. No further setup needed for most mods.

Troubleshooting

Game won't launch after installing

You likely installed the wrong architecture. Re-run the installer and try the other option (x86 vs x64).

Crash on startup

Install or repair the .NET 6.0 Desktop Runtime.

Nothing showing up / mods not loading

Check the mod .dll is in Mods (not Plugins), and that the game was fully closed during install.

Uninstalling

  1. Close the game
  2. Delete version.dll and dobby.dll from the game's install folder
  3. Delete the MelonLoader folder
  4. (Optional, full clean removal) also delete Mods, Plugins, and UserData
